[SCOPE:s_method|description|拉格朗日乘数法]
[SCOPE:s_method_let|let_bind|拉格朗日乘数法变量]
[0] 若要求函数 $f(x_1,\dots, x_N)$ 在 $m$ 个约束条件
[1] \begin{equation}
g_i(x_1, \dots, x_N) = 0 \qquad (i = 1,\dots, M)~
\end{equation}
[2] 下的极值，可用以下的
[3] 拉格朗日乘数法
[4] 令拉格朗日函数为
[5] \begin{equation}
\Lambda(x_1,\dots, x_N) = f + \sum_{i=1}^M \lambda_i g_i~,
\end{equation}
[6] 其中 $\lambda_i$ 为未知任意常数。
[/SCOPE:s_method_let]
[7] 同时满足约束条件和
[8] \begin{equation}
\frac{\partial \Lambda}{\partial x_i} = 0 \quad (i = 1,\dots, N)~
\end{equation}
[9] 的点，就是函数 $f(x_1, \dots, x_N)$ 的
[10] 极值点
[11] 稳定点
[12] 注意式 3 有 $N$ 条等式，结合式 1 共有 $M+N$ 条等式组成方程组（注意未必是线性的）。
[13] 而未知量 $x_1,\dots,x_N$ 和 $\lambda_1,\dots,\lambda_M$ 也有 $M+N$ 个。
[14] 所以一般情况下存在唯一解或有限个解。

[/SCOPE:s_method]
[SCOPE:s_ex1_geo|description|几何理解]
[SCOPE:s_example1|description|例1]
[15] 下面举一个二元函数的例子以说明拉格朗日乘数法的几何意义。
[16] 【图解】图 1：例 1

[SCOPE:s_ex1_let|let_bind|例1变量]
[17] 求函数 $f(x,y) = -a(x^2 + y^2)$ 在轨迹 $x+y = c$ 约束下的最值（$a,c$ 均为常数）。
[/SCOPE:s_ex1_let]
[18] 从几何上来理解，函数 $f(x,y)$ 是一个倒置的抛物面，其等高线的俯视图如图 1 蓝色的箭头代表函数值增加最大的方向，
[19] 即梯度的方向。
[20] 考察点沿着同一条等高线的位置变化时，函数值不变。
[21] 曲面上任意一点的梯度为
[22] \begin{equation}
\boldsymbol\nabla f = \frac{\partial f}{\partial x} \hat{\boldsymbol{\mathbf{x}}} + \frac{\partial f}{\partial y} \hat{\boldsymbol{\mathbf{y}}} = -2ax \hat{\boldsymbol{\mathbf{x}}} -2ay \hat{\boldsymbol{\mathbf{y}}} = -2a \boldsymbol{\mathbf{r}} ~,
\end{equation}
[23] 约束条件如图 1 中红线所示，我们要求的就是红线上面函数 $f$ 的极值。
[24] 在该题的情况下，所求的极值显然在红线离原点最近的地方出现。
[25] 但是如何总结出约束条件下多元函数极值点的一般的判断条件并用梯度表示呢？
[26] 根据梯度的性质（式 5 ），如果考察点沿红线的切向移动 $ \,\mathrm{d}{ \boldsymbol{\mathbf{l}} } $，函数值的微分为
[27] \begin{equation}
\,\mathrm{d}{f} = \boldsymbol\nabla f \boldsymbol\cdot \,\mathrm{d}{ \boldsymbol{\mathbf{l}} } ~.
\end{equation}
[28] 导数和极值的关系
[29] \begin{equation}
\boldsymbol\nabla f \boldsymbol\cdot \,\mathrm{d}{ \boldsymbol{\mathbf{l}} } = 0~.
\end{equation}
[30] \begin{equation}
g(x, y) = x+y-c = 0~,
\end{equation}
[31] 矢量共线的定义
[32] \begin{equation}
\boldsymbol\nabla f + \lambda \boldsymbol\nabla g = 0 \qquad \text{即} \qquad \boldsymbol\nabla (f + \lambda g) = 0~,
\end{equation}
[33] \begin{equation} \begin{aligned}
0 &= \frac{\partial}{\partial{x}} (f + \lambda g) = -2ax + \lambda~,\\
0 &= \frac{\partial}{\partial{y}} (f + \lambda g) = -2ay + \lambda~.
\end{aligned} \end{equation}
[34] 以上的推导可以很容易地拓展到多元函数和多个限制条件中去。
[35] 例如求三元函数 $f(x,y,z)$ 中（可看作空间的标量场），在两个约束条件 $g_1(x,y,z) = 0$，$g_2(x,y,z) = 0$ 下的极值。
[36] 注意两个约束条件分别代表一个曲面，同时成立代表两个曲面相交的空间曲线。
[37] 我们要在该曲线上求函数 $f$ 的极值。
[38] 同样，若曲线上的某点是极值，那必须满足曲线在该点的切线方向 $f$ 的方向导数为零，
[39] 即 $ \boldsymbol\nabla f$ 与该点法向量共线。
[40] 然而这是个三维问题，空间曲线的任意一点可以有两条不共线的法向量（不一定互相垂直），该点的任何其他法向量可以由这两个法线的线性组合来表示。
[41] 即存在常数 $\lambda_1$ 和 $\lambda_2$，使得
[42] \begin{equation}
\boldsymbol\nabla f + \lambda_1 \boldsymbol\nabla g_1 + \lambda_2 \boldsymbol\nabla g_2 = 0~.
\end{equation}
[43] \begin{equation}
\boldsymbol\nabla (f + \lambda_1 g_1 + \lambda_2 g_2) = 0~,
\end{equation}
[44] 其中 $f + \lambda_1 g_1 + \lambda_2 g_2$ 就是拉格朗日函数。
[45a] 当然，限制条件的数目不一定是 2，若只有一个限制条件 $g_1 = 0$
[45b] 则是求曲面 $g_1 = 0$ 上函数 $f$ 取得极值的点，条件是函数的梯度 $ \boldsymbol\nabla f$ 与曲面的法向量 $ \boldsymbol\nabla g_1$ 共线。

[/SCOPE:s_ex1_geo]
[/SCOPE:s_example1]
[SCOPE:s_example2|description|例2]
[SCOPE:s_ex2_let|let_bind|例2变量]
[46] 椭圆方程为 $x^2/a^2 + y^2/b^2 = 1$，求其内接长方型的最大面积（长方形的边平行于 $x$ 轴和 $y$ 轴）。
[47] 令长方型的右上角与椭圆的切点为 $(x,y)$，
[48] 则长方型的面积为 $S = 4xy$。
[49] 约束函数为 $g = x^2/a^2 + y^2/b^2 - 1$。
[/SCOPE:s_ex2_let]
[50] 拉格朗日函数为
[51] \begin{equation}
\Lambda(x,y) = S + \lambda g = 4xy + \lambda \left(\frac{x^2}{a^2} + \frac{y^2}{b^2} - 1 \right) ~.
\end{equation}
[52] \begin{equation}
\begin{cases}
4y + 2\lambda x/a^2 = 0\\
4x + 2\lambda y/b^2 = 0
\end{cases}~,
\end{equation}

[/SCOPE:s_example2]
[SCOPE:s_mechanics|description|力学意义]
[SCOPE:s_mech_let|let_bind|力学变量]
[53] 如果从力学角度审视约束下质点运动的稳定位置，拉格朗日乘数法将会显得十分自然。1
[54] 考虑一个特例:平面 $R^2$ 内一条曲线 $C:f(x,y)=0$ 上的轨道，轨道上有一个质点,f 可微。
[55] 力场为势场(例如，引力场)，忽略摩擦、空气阻力等非理想条件。
[56] 显然，轨道对质点的约束力 $R$ 必沿法线方向。
[57] 由于曲线 $C$ 的法向量有形式 $( \partial f/\partial x , \partial f/\partial y )$，由向量共线的定义，我们有理由设约束力为
[58] \begin{equation}
\boldsymbol{\mathbf{R}} = \left(\lambda \frac{\partial f}{\partial x} ,\lambda \frac{\partial f}{\partial y} \right) ~.
\end{equation}
[/SCOPE:s_mech_let]
[59] 既然力场是势场，
[60] 那么我们可以将力表示为势的梯度，
[61] 即
[62] \begin{equation}
\boldsymbol{\mathbf{F}} = - \boldsymbol\nabla p = \left(- \frac{\partial p}{\partial x} ,- \frac{\partial p}{\partial y} \right) ~.
\end{equation}
[63] $p=p(x,y)$ 是 $(x,y)$ 处的势能。
[64] 当势能取极值时，质点处于平衡位置（合外力 $ \boldsymbol{\mathbf{R}} + \boldsymbol{\mathbf{F}} =0$ 的位置），换言之，
[65] \begin{equation}
\boldsymbol{\mathbf{R}} + \boldsymbol{\mathbf{F}} = 0 ~.
\end{equation}
[66] 写成关于分量的方程组
[67] \begin{equation}
\lambda \frac{\partial f}{\partial x} - \frac{\partial p}{\partial x} = 0~, \quad
\lambda \frac{\partial f}{\partial y} - \frac{\partial p}{\partial y} = 0 ~.
\end{equation}
[68] 显然这正是拉格朗日乘数法的形式。
[/SCOPE:s_mechanics]
